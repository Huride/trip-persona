# TripPersona Demo Script

## Demo 1: Cafe Gallery

Input: `sample:cafe-gallery`

Survey:
- Travel window: 이번 봄
- Trip length: 1박 2일
- Destination: 추천받기
- Budget: 중간
- Companion: 혼자
- Pace: 여유롭게
- Walking: 1만 보 이하
- Include: 카페, 전시, 사진
- Avoid: 혼잡

Expected:
- Persona: slow urban cafe/gallery traveler
- Likely destinations: 서울, 도쿄, 가마쿠라
- Itinerary should emphasize cafes, galleries, quiet streets.
